LEGION 1.0

We are so many small.

LEGION is a decentralized, personal data system. Each base is a self-contained unit.
No cloud, no gatekeepers, no surveillance.

Each user is a node.
Each base lives in a folder.
Each folder contains rules, rights, and APIs.

You don't upload to a server — you share a key.
You don't ask permission — you publish a cell.
Your data isn't synced — it's yours.

LEGION is for developers, tinkerers, and believers in small tech.
It's not just a tool. It's a mindset.
